​ 软件：nasm 安装：sudo apt-get install nasm

2、开始操作

1、创建boot.asm文件 命令：sudo nano boot.asm

​ 输入以下代码：

​ 保存退出：Ctrl+x

2、将boot.asm编译为二进制文件boot.bin

​ 命令：nasm boot.asm -o boot.bin

​ 备注：二进制文件查看器：xxd与od

​ 操作方式：xxd 参数 文件名

​ od 参数 文件名

3、创建软盘，并将二进制boot.bin文件写入软盘

​ 命令：

​ a. 制作软盘镜像文件diska.img

​ dd if=/dev/zero of=diska.img bs=512 count=2880

​ b. 复制boot.bin到boot.img中

​ dd if=boot.bin of=boot.img bs=512 count=1

​ c. 复制diska.img中从第512字节往后的内容添加到boot.img中（这是dd最出彩的部分）

​ dd if=diska.img of=boot.img skip=1 seek=1 bs=512 count=2879
